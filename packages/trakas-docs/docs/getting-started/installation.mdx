Trakas is designed as a [monorepo](https://en.wikipedia.org/wiki/Monorepo)
with multiple [packages](https://github.com/trakas/trakas/tree/main/packages)

You can install Trakas with [npm](https://www.npmjs.com), [Yarn](https://yarnpkg.com), or a good ol' `<script>` via [unpkg.com](https://unpkg.com).

## JavaScript

<Tabs
  groupId={groupPackage}
  values={packageTabs}
  defaultValue={defaultPackage}
>

<TabItem value="npm">

```bash
npm install @trakas/js
```

</TabItem>

<TabItem value="yarn">

```bash

```

</TabItem>

<TabItem value="pnpm">

```bash

```

</TabItem>

</Tabs>
